# Лендинг аукциона «Оно тебе надо»

Верстка лендинга для аукциона вещей, в которые никто не верил. Проект выполнен в рамках обучения веб-разработке.
https://github.com/KateSarycheva/ono-tebe-nado-ad
## Технологии

- HTML5
- CSS3 (Flexbox, Grid)
- Методология БЭМ
- Адаптивная верстка
- Шрифт Raleway

## Особенности реализации

- Семантическая верстка
- Валидный HTML/CSS
- Кроссбраузерность (Chrome, Firefox, Yandex Browser)
- Pixel Perfect верстка
- Оптимизированные изображения
- Переиспользуемые CSS-классы
